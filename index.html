Javascript Flashcards

JavaScript

JavaScript is a programming language used for adding interactivity to websites.

Operators

Addition, subtraction, multiplication & division

Variables

A way to save data values to a certain name, which can be changed at anytime.

Variable Naming Conventions

A short statement that explains what the variable is. ie. "givenName"

Functions

A block of code designed to perform a particular task

Methods

Actions that can be performed on objects.

Arguments

An Array-like object accessible inside functions that contain the values of the arguments passed into that function

Parameter

The variables listed in the function definition

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining Methods

Calling one method after another on an object, in one continuous line of code.

Strings

Stores a series of characters, instead of numbers.

Booleans

A data value that is saved as true or false.

Undefined

A variable with no function or value

NaN

Not a Number

Escape

Will encode a string

Alert

When used, a popup will apear with an output displayed

Comments

Notes that don't affect how your code runs.s

JQuery

A JavaScript Library

Attributes

The value of a property, and can define how the property can be accessed.
